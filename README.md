### Usage

1. Run `npm install` or `yarn install`.
2. Run `yarn start` and access `http://localhost:3000`

- [ReactJS](https://reactjs.org/docs/) - The web framework used
- [Styled Components](https://styled-components.com/docs) - CSS in JS
